# webpack-boilerplate
Basic webpack 2.0 boilerplate

## Getting Started
See below

## Instalation
```npm install or yarn```

## Usage
Runs basic webpack command
```
build: webpack
```

Runs webpack in production mode
```
build-prod: webpack -p
```

Runs webpack dev server
```
server: webpack-dev-server
```

Runs browsersync
```
server-sync: webpack --watch
```
